================================================================================
  AI 路演教练与访谈复盘系统 — 同事使用说明（纯净交付包）
================================================================================

一、环境要求
  · Windows 10/11，已安装 Python 3.10 或以上（安装时勾选 Add Python to PATH）。
  · 可访问互联网（首次需 pip 安装依赖；运行时需调用云端转写与打分 API）。

二、第一次使用
  1. 将整个「AI路演教练_纯净交付版」文件夹放到任意路径（如桌面或 U 盘）。
  2. 双击「一键启动系统.bat」，等待依赖安装完成，浏览器会自动打开控制台。
  3. 在左侧边栏「首次使用请配置 API 密钥」中填写：
       · 阿里云 API Key（百炼 DashScope，用于转写）
       · DeepSeek API Key（用于复盘打分）
     点击「保存并测试连接」，直至两项均为绿灯。
  4. 根目录会自动生成或更新 .env 文件保存密钥（请勿把含真实 Key 的文件夹上传到公共网盘）。

三、日常操作
  · 选择业务大类、填写项目/批次名称，按需上传 QA 参考文件与音频。
  · 点击「开始生成批量复盘报告」，报告会保存到您填写的「数据归档根目录」下。

四、常见问题
  · 若提示找不到 python：请重新安装 Python 并勾选加入 PATH，或联系 IT。
  · 若端口被占用：关闭其它 Streamlit 窗口或更换端口（命令行：streamlit run app.py --server.port 8502）。
  · 转写主通道可能使用硅基流动等其它 Key，本纯净包以侧边栏配置的阿里云与 DeepSeek 为必选项。

五、技术支持
  · 本包不含测试数据、不含 output 样例；首次运行请在界面内完成密钥自检。
